## О проекте

**Simple Android Math Tasks App** — это минималистичное Android‑приложение, которое генерирует простые арифметические задания и позволяет пользователю решать их в интерактивной форме.

Проект создан с упором на:

* понятную структуру Android‑приложения
* чистую логику без избыточных зависимостей
* простоту расширения и модификации

---

## Возможности

* Генерация арифметических заданий со случайными значениями
* Проверка введённых пользователем ответов
* Мгновенная обратная связь
* Минималистичный пользовательский интерфейс

## Установка и запуск

```bash
git clone https://github.com/valmork/Simple-android-math-tasks-app.git
```

1. Откройте проект в Android Studio
2. Дождитесь завершения синхронизации Gradle
3. Запустите приложение на эмуляторе или физическом устройстве
